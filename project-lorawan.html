LoRaWAN Asset Tracker

Smart, long-range tracking for industrial assets.

Aug 2023 - Jan 2025 LoRaWAN, IoT, Embedded Systems

Overview

This device tracks industrial equipment over 5km+, even without GPS. It's battery-efficient, lasting 12–18 months on a single charge, and built to survive harsh environments.

Highlights

  • 5km+ range in cities
  • 12–18 months battery life
  • GPS + BLE positioning
  • Tamper and shock alerts
  • -40°C to +85°C operating range
  • Cloud-based location tracking with geofencing

Technical Implementation

Hardware

  • STM32L072CZ microcontroller
  • Semtech SX1276 LoRa radio
  • Custom low-power PCB
  • 3-axis accelerometer for motion
  • IP67 waterproof casing

Firmware

  • Zephyr RTOS-based power management
  • Adaptive data transmission
  • Encrypted (AES-128) LoRaWAN communication
  • Over-the-air (OTA) firmware updates

Cloud Integration

  • The Things Network (TTN) backend
  • Node-RED data workflows
  • MongoDB for historical data storage
  • React dashboard for live asset tracking
