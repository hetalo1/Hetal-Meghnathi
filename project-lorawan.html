LoRaWAN Asset Tracker

Long-range IoT tracking solution for industrial assets

Duration: Aug 2023 - Jan 2025

Technologies: LoRaWAN, Embedded Systems, IoT

Project Overview

The LoRaWAN Asset Tracker is a low-power, long-range solution designed for monitoring industrial equipment across large facilities. It provides real-time location tracking with over 12 months of battery life, even in GPS-denied environments.

Key Features

  • 5km+ range in urban areas
  • 12-18 month battery life
  • Hybrid GPS + BLE positioning
  • Shock and tamper detection
  • Operates from -40°C to +85°C
  • Cloud dashboard with geofencing

Technical Implementation

Hardware Design

  • STM32L072CZ Microcontroller
  • Semtech SX1276 LoRa Transceiver
  • Custom Ultra-Low-Power PCB
  • 3-axis Accelerometer
  • IP67 Rated Enclosure

Firmware

  • Zephyr RTOS for power management
  • Adaptive transmission intervals
  • Over-the-air firmware updates (OTA)
  • Encrypted communication (AES-128)

Cloud Integration

  • TTN Network Integration
  • Node-RED Data Processing
  • MongoDB Storage
  • React.js Based Dashboard
