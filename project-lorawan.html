LoRaWAN Asset Tracker

Industrial IoT tracking across challenging environments

Duration: Aug 2023 – Jan 2025

Technologies: LoRaWAN, Embedded Systems, IoT, Zephyr RTOS

Project Overview

Developed an ultra-low-power asset tracker leveraging LoRaWAN technology, ensuring efficient real-time monitoring of industrial equipment even in GPS-denied environments. Designed for robust field deployment, with optimized firmware and cloud integration.

Key Features

  • 5km+ communication range (urban)
  • Battery life: 12–18 months
  • GPS + BLE hybrid positioning
  • Shock & tamper detection
  • Temperature tolerance: -40°C to +85°C
  • Secure cloud dashboard integration

Technical Implementation

Hardware Design

  • STM32L072 MCU + SX1276 LoRa Transceiver
  • Low-power custom PCB layout
  • 3-axis accelerometer
  • Sealed IP67 enclosure

Firmware

  • Built on Zephyr RTOS
  • Adaptive transmission intervals
  • Secure AES-128 communication
  • OTA (Over-the-Air) firmware updates

Cloud Backend

  • The Things Network integration
  • Node-RED workflows
  • MongoDB storage
  • React-based data dashboard
