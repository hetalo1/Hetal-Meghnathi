Smart Nursing Assistant

Academic Project | Developed Individually

Project Overview

This project was developed as part of my academic curriculum, where I designed and implemented a smart nursing assistant system from scratch. The goal was to create a proof-of-concept for automated patient monitoring that could assist healthcare professionals.

Technical Implementation

I single-handedly developed the entire system including:

  • Hardware Design: Custom PCB with STM32 microcontroller, sensor array, and wireless communication modules
  • Firmware Development: Bare-metal C programming for real-time sensor data acquisition and processing
  • Algorithm Design: Implemented basic anomaly detection algorithms for vital signs monitoring
  • Prototype Construction: 3D-printed enclosure and mechanical components
  • Testing & Validation: Conducted extensive testing with simulated patient data

Key Achievements

  • Successfully demonstrated all core functionalities within project timeline
  • Achieved 92% accuracy in basic anomaly detection
  • Implemented power optimization extending battery life to 72+ hours
  • Received top marks in academic evaluation

Technologies Used

Embedded C STM32 Sensor Fusion BLE KiCAD 3D Printing
