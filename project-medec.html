MedEc Smart Nurse

AI-integrated smart nursing robot

Project Overview

The MedEc Smart Nurse is an autonomous robotic assistant designed for post-operative patient monitoring in hospital environments. The system integrates real-time vital sign monitoring with AI-powered analysis to detect potential complications early.

My role involved developing the embedded control system, sensor integration, and communication protocols between the robot and central nursing station.

Key Features

  • Real-time vital sign monitoring (heart rate, blood pressure, oxygen levels)
  • AI-powered anomaly detection with 95% accuracy
  • Autonomous navigation in hospital environments
  • Secure patient data transmission
  • Emergency alert system

Technologies Used

Embedded C++ ROS STM32 TensorFlow Lite BLE LoRa

Project Details

Client: MedEc Healthcare Solutions

Duration: 8 months

Team Size: 5 engineers
