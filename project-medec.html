MedEc Smart Nurse

AI-integrated smart nursing robot for post-operative care

Jan 2021 - Aug 2023 Embedded Systems, IoT, Healthcare Robotics

Project Overview

The MedEc Smart Nurse is an autonomous robotic assistant designed to revolutionize post-operative patient monitoring in hospital environments. The system integrates real-time vital sign monitoring with AI-powered analysis to detect potential complications early, reducing nurse workload and improving patient outcomes.

Key Features

  • Continuous monitoring of vital signs (ECG, SpO2, temperature)
  • Autonomous navigation between patient beds
  • AI-powered anomaly detection with 94% accuracy
  • Multi-protocol wireless communication (BLE/LoRa/WiFi)
  • Medication reminder and delivery system

Technical Implementation

Hardware Architecture

  • STM32H743VI as main controller
  • Raspberry Pi 4 for AI processing
  • Custom PCB for sensor integration
  • LIDAR and ultrasonic sensors for navigation

Software Stack

  • Embedded C++ for real-time control
  • ROS (Robot Operating System) for navigation
  • TensorFlow Lite for edge AI
  • Custom RTOS for sensor fusion

Communication

  • BLE for wearable sensors
  • LoRa for long-range hospital-wide alerts
  • WiFi for high-bandwidth data transfer
  • MQTT protocol for cloud integration

Challenges & Solutions

Real-time Sensor Fusion

High Complexity

Problem: Integrating data from multiple sensors (ECG, SpO2, temperature) with varying sampling rates while maintaining real-time performance.

Solution: Developed a custom RTOS scheduler with priority-based task management and implemented a Kalman filter for sensor fusion, reducing CPU load by 40%.

Power Management

Medium Complexity

Problem: Achieving 8+ hours of battery life while powering multiple sensors and wireless modules.

Solution: Designed a dynamic power management system that adjusts CPU frequency and selectively powers peripherals based on usage patterns, extending battery life by 65%.

Wireless Interference

High Complexity

Problem: Hospital environments have significant RF interference from medical equipment.

Solution: Implemented adaptive frequency hopping for BLE and developed a hybrid communication protocol that automatically switches between LoRa and WiFi based on signal quality.
